# PhotoShow · 图秀
**PhotoShow** is a minimalist self-hosted image gallery powered by Cloudflare R2. Upload, organize, and display your images with grid/masonry views, category management, and password-protected uploads. No database required.

**图秀** 是一款基于 Cloudflare R2 的轻量级自建图片的存储、展示程序。使用图秀，可以构建一个提供网格/瀑布流浏览、分类管理、密码保护上传等核心功能，无需数据库，部署简单的网站。

**PhotoShow** is a lightweight, self-hosted image gallery that stores your pictures on **Cloudflare R2** (S3-compatible object storage). It offers a clean, responsive interface with grid/masonry layout, category management, and password-protected uploads.

🔗 **Live Demo:** [https://photoshow.cn](https://photoshow.cn)

---

## ✨ Features

- **📸 Image Display** — Grid & masonry views with lazy loading and pagination (20 images per page)
- **☁️ Cloudflare R2 Storage** — All images stored securely on Cloudflare R2
- **📁 Category Management** — Organize images into folders (e.g., girls, arts, city, wildlife)
- **🔒 Password-Protected Upload** — Upload images locally or via URL with admin password verification
- **🖼️ Lightbox Preview** — Click any image to open a full-screen lightbox with keyboard navigation (← → ESC)
- **📱 Fully Responsive** — Works seamlessly on desktop, tablet, and mobile devices
- **⚡ Fast & Minimal** — No database required, just PHP and Cloudflare R2
